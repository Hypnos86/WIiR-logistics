#### Aplikacja na użytek Komendy Wojewódzkiej Policji w Poznaniu WIiR

#### Biblioteki
- Python 3.11
- Django 5.0
- Postgresql
- Bootstrap 5.0.2

1. Instalacja bibliotek: 
`python -m pip install -r requirements.txt`
1. Odpalenie aplikacji: 
`python manage.py runserver`
